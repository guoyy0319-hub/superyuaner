# Feature selection of scheduling rules according QS-NSGAⅡ
本库里包含一些我的调度特征选择算法的实现和可视化
在强化学习模型中，若直接以随机初始化的图结构和策略进行强化学习训练，往往会出现前期探索不稳定、收敛速度慢等问题。为此，引入经典启发式调度规则作为先验知识（Prior Rules），通过非支配排序遗传算法II（NSGA-II）的变种QS-NSGAⅡ在多目标优化框架下生成各种小规模柔性作业车间调度问题案例的调度方案，自动筛选出对系统性能贡献最大的规则组合，作为强化学习策略的初始拓扑概率分布，并将其转换为每一决策步的MDP，作为专家经验，以提升训练效率与稳定性。
算法参考：https://link.cnki.net/urlid/11.4762.TP.20250918.1006.012.
